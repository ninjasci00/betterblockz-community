## Contributing to BetterBlockZ Community

This repository is for **translations only**.  
If your contribution is not a language file, it does not belong here.

## Allowed Contributions
- New language files
- Fixes or improvements to existing translations

## Not Allowed
- Mod code or assets
- Docs, changelogs, feature requests, or bug reports  

## File Rules
- Place files in `assets/betterblockz/lang/`
- `en_us.json` is the **source of truth**
- Other languages must match its keys exactly

## How to Contribute
1. Fork this repo
2. Copy `en_us.json`
3. Rename to your locale (`es_es.json`, `ja_jp.json`, etc.)
4. Translate **values only** (do not change keys)
5. Ensure valid JSON
6. Submit one language per pull request

## Strict Rules
PRs will be rejected if they:
- Add/remove/rename keys
- Change formatting, punctuation, or capitalization unnecessarily
- Modify placeholders (`%s`, `%d`, `{0}`, `{1}`) or color codes (`§a`, `§6`)
- Contain low‑effort or undisclosed machine translations
